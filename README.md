# Vitrine Nonowarwood

Petit site vitrine personnel pour présenter les edits, vidéos et petits sites web de Nonowarwood.

## Description

- Page d'accueil responsive en HTML/CSS avec quelques animations légères.
- Présentation de projets vidéo, liens vers les réseaux sociaux et travaux web.
- Design inspiré de la culture K-pop et d'une direction artistique typographique.

## Structure du projet

- `index.html` : page principale (liens, accueil, moodboard ?).
- `portfolio.html` : page portfolio détaillée avec sections À propos, Edits, Sites web et Réseaux.
- `background/` : images de fond et visuels utilisés dans le site.
- `edits/` : vidéos et fichiers médias des edits.
- `links/` : dossier contenant la page de liens.
- `inspi/` : éléments d'inspiration et ressources visuelles.

## Utilisation

1. Ouvrir `portfolio.html` dans un navigateur pour voir le site.
2. Modifier le contenu directement dans le HTML ou le CSS intégré si besoin.

## À améliorer

- Ajouter une page de contact ou formulaire.
- Séparer le CSS dans un fichier dédié pour faciliter la maintenance.
- Optimiser les médias (images et vidéos) pour un chargement plus rapide.

## Remarques

- Le site est principalement en français.
- Les vidéos et images sont chargées localement depuis les dossiers `background/` et `edits/`.
